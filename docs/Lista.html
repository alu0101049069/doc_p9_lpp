    insertar(value) click to toggle source

    se inserta un elemento en la lista

    • se crea un nuevo nodo con el valor a insertar

    • se comprueba si la cabeza y la cola están vacías y si es así se crea en ellas el nodo

    • se declara el previo del nodo creado como la cola

    • el siguiente nodo de la cola será el nuevo nodo

    • la nueva cola pasará pues a ser el nuevo nodo

    • se incrementa la longitud de la lista

    # File lib/comida/lista.rb, line 63
def insertar(value)
    new_node = Node.new(value: value) 

    
    self.cabeza = new_node if cabeza.nil?
    self.cola = new_node if cola.nil?
  
    new_node.prev = self.cola

    
    self.cola.sig = new_node

    self.cola = new_node
    @length += 1
end
